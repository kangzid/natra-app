# NATRA Mobile App

Mobile application for NATRA HRIS system built with Capacitor and Vanilla JS.

## 📁 Dokumentasi
- [Panduan Pemeliharaan (Maintenance)](docs/MAINTENANCE.md)
- [Pengelolaan Aset & Logo](docs/ASSET_MANAGEMENT.md)
- [Arsitektur Sistem](architecture_guide.md)

## ⚡ Quick Start
1. Install dependencies: `npm install`
2. Run development: `npm run dev`
3. Build for Mobile: `npm run build && npx cap sync android`

## 📱 Features
- Real-time Location Tracking
- Attendance System (Check-in/Out)
- Task Management
- iOS Style Premium UI
- Persistent Dark Mode
